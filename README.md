## Database and API Functionality

Database contains two tables
1. User
    * id (Primary Key)
    * real_name
    * tz
2. ActivityPeriod
    * id
    * start_time
    * end_time
    
If user already exists, then only `ActivityPeriod` table is updated. Otherwise, both `User` and `ActivityPeriod` tables are updated
